# Ascii-Converter

Getting Started:
- Install Python
- Install FFMPEG
- Get Math to FFMPEG using `which ffmpeg` 
- Update FFMPEG path 
- Run code using `[command for img_to_ascii]` or '[command for vid_to_ascii]'
